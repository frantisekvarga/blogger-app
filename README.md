### Template of simple Full Stack App

## Structure

- `docs` - generic documentation
- `libs` - shared libraries
- `apps` - different applications
  - [`api`](./apps/api) - API layer
  - [`app`](./apps/app) - App frontend

## Requirements

- Bash (or WSL)
- Node 20
- Yarn 1
- Nx ^19.1.1
